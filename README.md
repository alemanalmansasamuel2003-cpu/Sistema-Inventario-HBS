# 📦 Sistema Inventario HBS

Aplicación móvil desarrollada para el **Hogar El Buen Samaritano**, diseñada para gestionar el inventario de productos de forma eficiente mediante una interfaz intuitiva y un backend robusto.

---

## 🚀 Características

- 🔐 Inicio de sesión con autenticación.
- 👤 Gestión de usuarios (Administrador y Encargado).
- 📦 Registro de productos.
- ✏️ Edición y eliminación de productos.
- 🔍 Búsqueda rápida de productos.
- 📉 Reporte de productos con stock bajo.
- 📅 Reporte de productos próximos a vencer.
- 📊 Panel de reportes del inventario.

## ⚙️ Instalación

### Clonar el repositorio

```bash
git clone https://github.com/alemanalmansasamuel2003-cpu/Sistema-Inventario-HBS.git
```

### Instalar dependencias del backend

```bash
npm install
```

### Ejecutar el servidor

```bash
node app.js
```

### Ejecutar la aplicación móvil

```bash
cd inventario_app
npm install
npx expo start
```
